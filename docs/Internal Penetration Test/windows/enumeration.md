# Windows Enumeration Tools

윈도우 침투테스팅에서 사용할 수 있는 여러가지 오픈소스툴 들이 있습니다. 침투 테스팅을 진행 할때, 항상 명시된 Scope를 절대적으로 지켜야 하며, 해당 툴들이 어떻게 동작하는지 이해하고 Eumeration 툴을 사용을 하여야 합니다.
